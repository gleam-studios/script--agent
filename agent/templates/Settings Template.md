# Settings Template (设定集)

> **定位**：项目全局资产注册表。所有在后续「分集大纲」与「分集剧本」中出现的人物、物品、场景**必须**先在本设定集中以 `∆分类 + @名称` 格式注册，方可在下游引用。
>
> **资产标记体系（硬规则）**：
>
> - `∆` 为**资产分类标记**，仅三种：`∆人物`、`∆物品`、`∆场景`。
> - `@` 为**具体资产名称标记**，格式 `@中文名(英文名)` 或 `@中文名`。
> - 名称一经注册，后续所有 STAGE 6/7 引用**必须**与此处完全一致（含括号内英文），一字不差。
> - 新资产须先在本设定集中追加注册，再在大纲/剧本中引用。
>
> **来源**：本设定集的初始草稿可由 LLM 从 STAGE 1-4 产物中自动提取，用户审核补充后确认。
